# hudson_plot

hudson_plot是基于python3开发的一款轻量级GUI工具，支持动态设置震源机制散点图样式，调节颜色、大小、标记类型等参数。

✅ 支持读取 .dat 震源机制数据文件
✅ 可视化绘制 Hudson 图框架
✅ 支持交互式设置：• 点类型（如 o、x 等）• 点颜色（红、蓝、绿）• 点大小滑块调节
